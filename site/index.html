--csstyle
Clean, Simple Styling for the Web.

With csstyle you organize your CSS as reusable components

@include component(button) {
  background: #6BD9ED;
  padding: 10px;
}
<span class="button">Neat!</span>
Neat!

Make your components super flexible with options

@include component(button) {
  background: #6BD9ED;
  padding: 10px;

  @include option(action){
    background: #A7E040;
    border: 2px solid #626262;
  }
}
<span class="button --action">Neat!</span>
Neat!

Go ahead and try it! Add another option, for example "warning" or "confirm".
